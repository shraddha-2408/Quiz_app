# Quiz_app

A simple and interactive Quiz Application built using HTML, CSS, and JavaScript. This project allows users to answer multiple-choice questions, get instant feedback, and view their final score.

## Features

* Interactive multiple-choice quiz
* Instant answer validation
* Score calculation
* Responsive and user-friendly interface
* "Play Again" functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```
Quiz-App/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/your-username/simple-quiz-app.git
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
